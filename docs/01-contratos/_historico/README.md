# Histórico de contratos

Espacio reservado para contratos o acuerdos previos que **ya no rigen el trabajo actual** del proyecto NeoSer.

## Cuándo poner algo aquí

- El acuerdo lo emitió un equipo anterior que ya no participa en el proyecto (por ejemplo, etapas previas con terceros que dejaron de colaborar).
- El alcance del documento fue reemplazado por uno nuevo (ver `../README.md`).
- Existe valor de trazabilidad pero no se usa como referencia operativa.

## Reglas

- Cada documento histórico debe ir acompañado de una nota breve (puede ser una línea en este README) explicando: (a) qué fue, (b) cuándo dejó de aplicar, (c) qué documento lo reemplazó.
- Nunca borrar — preservar siempre el archivo original.
