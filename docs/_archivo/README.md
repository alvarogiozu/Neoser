# Archivo histórico

Esta carpeta preserva archivos que **ya no se usan en el trabajo activo** pero se mantienen por trazabilidad. **No borrar.**

## Qué hay aquí y por qué

| Archivo | Por qué se archivó |
|---|---|
| `prompt-para-crm.md` | Archivo creado vacío durante una sesión previa. Se preserva por si tenía intención conocida de retomarse. |
| `transcripcion-crm-video.md` | Placeholder de una transcripción que nunca se cargó. Se preserva para trazabilidad. |

## Reglas

- Si un archivo de esta carpeta se va a retomar, **moverlo a su carpeta operativa** (`00-contexto/`, `03-playbooks-revops/`, etc.) — no editarlo en sitio.
- Si se confirma que algo no aporta nada, abrir un PR con la justificación antes de borrar.
- Cualquier archivo nuevo que llegue aquí debe sumarse a la tabla de arriba con su motivo.
